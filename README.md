# youtube-kids

## Project setup
```
npm install
```

Make sure you have a `.env` with:
```
YOUTUBE_API_KEY=xxxx
```

This API key needs access to the YouTube Data Api (V3).

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
